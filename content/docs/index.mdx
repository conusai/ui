# ConusAI UI

ConusAI UI now ships with a first-class documentation surface inside the same Next.js 16 app as the demo.

## What is here

- A real docs route at `/docs`
- Search-backed component pages
- Interactive previews for core mobile and workspace primitives
- MDX content living beside the app, not in a separate site

## Why this setup

Fumadocs matches this stack directly: Next.js App Router, Tailwind 4, React 19, and a shadcn-style design system. That keeps the docs fast, local to the app, and easy to evolve with the actual components.
