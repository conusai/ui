# Mobile Footer

`MobileFooter` is the bottom navigation primitive for compact views where primary actions need to stay reachable during scroll.

<MobileFooterPreview />

## Usage

```tsx
import { Home, Settings } from "lucide-react";
import { MobileFooter } from "@conusai/ui";

const items = [
	{ id: "home", label: "Home", icon: Home },
	{ id: "settings", label: "Settings", icon: Settings },
];

<MobileFooter
	items={items}
	activeItem="home"
	onChange={(id) => console.log(id)}
/>
```

## Notes

- Best suited for 3 to 5 top-level destinations.
- Animation is reduced automatically when reduced-motion is enabled.
- The parent surface should reserve enough bottom spacing for safe-area insets.

## Public props

| Prop | Type | Description |
| --- | --- | --- |
| `items` | `FooterTab[]` | Navigation items with labels and icons |
| `activeItem` | `string` | Selected item identifier |
| `onChange` | `(id: string) => void` | Called when an item is selected |
| `className` | `string` | Optional style override hook |
