# Logistix AI Demo

This demo shows how the existing ConusAI UI primitives scale from a todo showcase into a more realistic finance workflow: intake, extraction, review, and re-invoicing.

<LogistixDemo />

## What it proves

- `Header`, `LeftSidebar`, `MobileFooter`, `MobilePreviewFrame`, and `Loader` all remain viable in a denser business workflow
- The now-extensible `RightSidebar` can host custom inspector content without losing the original todo-demo behavior
- AI screenshot exploration can target the new `/demo/logistix` route directly through the screenshot generator config
